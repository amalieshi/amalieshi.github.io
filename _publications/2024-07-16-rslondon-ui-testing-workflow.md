## Abstract

Presented at **RSLondon Southeast 2024**, held at the **Huxley Building, Imperial College London**, 180 Queen's Gate, South Kensington, London SW7 2RH. Tuesday 16 July 2024, 10:00–17:00.

This work presents a workflow for automating UI software testing and reviews its application in a commercial medical imaging case study. The approach leverages Python-based automation to emulate user interactions, validate test data, and generate automated reports, improving efficiency and reliability in medical software testing.
